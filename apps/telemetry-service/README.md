 ## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

# О проекте

**telemetry-service** — сервис, отвечающий за взаимодействие с телеметрией, опрашивает датчики и получает их значения.
